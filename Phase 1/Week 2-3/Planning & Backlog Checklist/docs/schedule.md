# Project Schedule — Flashcard Quiz App

## 15-Week Schedule

| Week | Goals | Deliverables |
|------|-------|-------------|
| 1 | Project setup, environment config | Repo created, React + Node.js initialized |
| 2 | Planning & backlog creation | Backlog, Sprint 1 plan, schedule, team roles |
| 3 | Risk management & team workflow | Risk register, PR/issue templates, updated backlog |
| 4 | QA & testing setup | QA plan, 5 unit tests, defect log, bug fix PR |
| 5 | Git mastery & SCM | Branching, merge conflict resolution, release notes, v0.5 tag |
| 6 | CI/CD pipeline setup | GitHub Actions workflow, automated tests on PR |
| 7 | Deployment & support | Deployed app (Vercel/GitHub Pages), deployment plan, support plan |
| 8 | Maintenance & refactoring | Tech debt list, refactor PR, performance notes, v0.8 tag |
| 9 | Sprint 2 — Quiz mode | Start Quiz (US-06), Self-Grade (US-07) |
| 10 | Sprint 2 continued | Quiz Score Summary (US-08), Shuffle (US-09) |
| 11 | Sprint 3 — Persistence & polish | Persistent Storage (US-10), UI improvements |
| 12 | Integration testing | End-to-end tests, cross-browser checks |
| 13 | User feedback & iteration | Collect feedback, fix issues, polish UI |
| 14 | Documentation finalization | User guide, API docs, final README |
| 15 | Final presentation & submission | Demo, final report, v1.0 release |

- **Week 5:** v0.5 — Core flashcard CRUD working (create, view, flip, edit, delete)
- **Week 8:** v0.8 — Refactored and optimized
- **Week 10:** v0.9 — Quiz mode fully functional
- **Week 15:** v1.0 — Final release with all features
